# 福牌·像素翻牌游戏 🎴

一个像素风格的九宫格翻牌游戏，需要输入祝福语才能翻牌。红色系渐变，最多同时翻开三张。

## ✨ 在线试玩
[点击这里访问游戏](https://你的用户名.github.io/fuyang-game)

## 🎮 游戏规则
1. 输入一句祝福语，点击【祝福·开牌】
2. 点击格子翻开卡片（每次翻牌消耗一次祝福）
3. 最多同时翻开3张卡片
4. 配对相同的卡片会保持翻开
5. 「福到了」是独牌，不会消除

## 🛠️ 本地运行
直接打开 `index.html` 即可游玩

## 📱 适配
支持手机端和电脑端
